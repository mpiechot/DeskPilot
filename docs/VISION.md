# DeskPilot Vision

DeskPilot is a local desktop control panel that helps the user switch between desktop workflows with minimal friction.

The first major problem is browser tab chaos.

The browser should no longer be used as an unsafe memory system for tasks, research, entertainment, projects and later reading.

DeskPilot should provide categories such as:

- Work
- Research
- Entertainment
- Projects
- Later / Inbox

The user should be able to:

- open a category as a browser session
- save the current browser window into a category
- close the saved browser window
- restore sessions reliably
- move unused tabs into sleep lists
- save long-term pages as categorized bookmarks
- reduce the number of active tabs

DeskPilot must work as a normal Windows desktop window first.

Later it should be usable on a small touch display connected to the PC.

Future extensions may include:

- lighting control
- audio control
- desk height control
- focus modes
- entertainment modes
- development modes

The product vision must not be changed by Codex.
