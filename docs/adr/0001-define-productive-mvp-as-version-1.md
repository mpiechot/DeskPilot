# Define Productive MVP As Version 1.0

DeskPilot 1.0 is the productive MVP: the current browser-session workflow must be safe, reliable, locally owned, and independent from ongoing development work. Larger extensions such as sleep lists, bookmark workflows, notifications, and deeper touch-display polish remain important, but they follow after 1.0 because the main reason to use the app already exists and must become trustworthy first.
