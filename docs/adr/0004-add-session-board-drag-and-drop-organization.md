# Add Session Board Drag-And-Drop Organization

DeskPilot should add an app-only Session Board where each category shows a compact scrollable list of its saved tabs. Users can drag saved tabs between category lists to move them and drag within a list to change tab order; restoring a category should respect that order. This keeps the Productive MVP focused on browser-session organization without expanding the browser extension scope. Drag-and-drop can be mouse-first for 1.0; deeper touch polish should follow after productive use is safe.
