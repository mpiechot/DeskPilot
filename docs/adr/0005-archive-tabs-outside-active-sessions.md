# Archive Tabs Outside Active Sessions

DeskPilot stores an optional archive timestamp on a Saved Tab. An Archived Tab remains in its original Category but is excluded from the active Browser Session, category tab counts, drag-and-drop ordering and browser restore. The Archive view can return it to the end of the active Session without creating a duplicate.

This keeps archiving distinct from soft deletion: Recovery remains for accidentally removed data, while Archive is an intentional cleanup workflow. Saving the same URL into the same Category again reactivates the archived row and updates its title instead of hiding the save as a duplicate or creating a second row.
